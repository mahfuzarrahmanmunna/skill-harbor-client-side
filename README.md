## 🎯 Purpose
An educational platform where users can browse, enroll in, and manage online courses securely.

## ⚙️ Tech Stack
- React
- TailwindCSS + DaisyUI
- Firebase Auth
- Node.js + Express
- MongoDB
- JWT

## 🌟 Key Features
- Firebase Auth + JWT
- Browse and enroll in courses
- View enrolled courses
- Protected routes
- Fully responsive design
